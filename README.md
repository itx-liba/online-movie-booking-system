Online Movie Booking System

Description

An online movie booking platform that allows users to browse movies, watch trailers, view reviews, and book movie tickets online.

Features

1. User Registration & Login
2. Movie Listings
3. Movie Reviews & Ratings
4. Trailer Viewing
5. Ticket Booking
6. Seat Categories (Gold, Platinum, Box)
7. Admin Dashboard
8. Theater Management
9. Movie Management
10. Show Schedule Management

Technologies Used

1. HTML
2. CSS
3. Bootstrap
4. JavaScript
5. PHP
6. MySQL
